## 1. 자막 위치 (`manual-subtitle-position`)

**평가 방법**: 영상을 전체 재생하며 자막이 화면 하단 1/4 이내에 위치하는지, 배경과 충분한 대비가 있는지 확인한다.

| 체크 | 설명 |
|------|------|
| [ ] PASS | 자막이 명확히 읽히고, 배경과 충분한 대비, 이탈 없음 |
| [ ] FAIL | 자막이 경계 밖으로 이탈하거나 배경에 묻힘 |

## 2. 한글 정확성 (`manual-korean-accuracy`)

**평가 방법**: 원고(스크립트)와 자막을 나란히 놓고 오자·띄어쓰기·발음 불일치를 기록한다.

| 체크 | 설명 |
|------|------|
| [ ] PASS | 오자 0건, 발음 불일치 최대 1건 |
| [ ] FAIL | 오자 1건 이상 또는 발음 불일치 2건 이상 |

## 3. 타이밍 동기화 (`manual-timing-sync`)

**평가 방법**: 자막 등장 시점과 음성 시작 시점의 차이를 5개 이상 구간에서 확인한다.

| 체크 | 설명 |
|------|------|
| [ ] PASS | 모든 확인 구간에서 ±300ms 이내 |
| [ ] FAIL | 500ms 이상 차이가 1건 이상 |

**확인한 구간 및 오차**:
| 구간 (초) | 추정 오차(ms) |
|-----------|--------------|
| | |

## 4. 전환 효과 (`manual-transition`)

**평가 방법**: 장면 전환 구간 전체를 집중 시청하며 갑작스러운 컷·플래시·화면 깜박임을 확인한다.

| 체크 | 설명 |
|------|------|
| [ ] PASS | 모든 전환이 부드럽고 갑작스러운 변화 없음 |
| [ ] FAIL | 갑작스러운 전환 또는 화면 깜박임 1건 이상 |

## 5. 음성 명료도 (`manual-voice-clarity`)

**평가 방법**: 헤드폰 없이 일반 스피커로 재생하며 내용 파악 가능 여부를 확인한다.

| 체크 | 설명 |
|------|------|
| [ ] PASS | 내용을 이해하는 데 지장 없음 |
| [ ] FAIL | 노이즈·왜곡으로 내용 파악 어려움 |

## 6. BGM 밸런스 (`manual-bgm-balance`)

**평가 방법**: 음성과 BGM의 상대 음량을 청취하며 BGM이 음성을 가리는지 확인한다.

| [ ] FAIL | BGM이 음성과 비슷하거나 더 크게 들림 |
