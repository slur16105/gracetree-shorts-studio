# YouTube Music 참고 화면 조정 기록

## 유지한 아이디어

- 왼쪽의 큰 집중 영역과 오른쪽 목록의 2분할 구조
- 어두운 표면 계층
- 하단에 지속적으로 보이는 현재 상태 영역
- 선택된 목록 행을 표면 색으로 구분하는 방식

## 제품에 맞게 변경한 아이디어

- 앨범 아트 영역 → 다중 파일 드래그앤드롭·입력 슬롯
- 트랙 목록 → 완료된 영상 목록
- 플레이어 → 최근 작업과 렌더링 단계·진행률
- 재생목록 탐색 → 홈·사용 가이드·설정의 최소 사이드바

## 제외한 아이디어

- 검색창, 좋아요, 싫어요, 셔플, 반복, 볼륨과 같은 음악 재생 전용 기능
- 완료 목록과 중복되는 별도 목록 화면
- 음악 서비스의 콘텐츠 추천·탐색 구조

## 참고자료 상태

사용자가 대화에 첨부한 화면을 확인해 결정에 반영했다. 전달된 로컬 Desktop 경로에서는 원본 파일을 찾지 못해 `imports/`에는 복사하지 않았다. 시각 계약은 `DESIGN.md`와 `EXPERIENCE.md`가 우선한다.
