첫 영상 만들기

게시 날짜를 정하고 준비한 파일을 한 번에 등록하면 GraceTree Shorts Studio가 파일을 분류하고 영상 생성 준비 상태를 확인합니다.

1

게시 날짜 선택

영상이 게시될 날짜를 선택하세요. 이 날짜가 작업 폴더와 완료 목록의 기준이 됩니다.

2

파일 한 번에 등록

썸네일, 음성, 스크립트를 드롭 영역에 함께 놓으세요. 배경음악은 선택 사항입니다.

3

준비 상태 확인

`100% · 필수 입력 4/4`가 표시되면 오른쪽의 영상 생성 버튼을 누르세요.

권장 파일 구성

썸네일thumbnail.png · PNG 또는 JPG 음성 녹음voice.m4a · M4A 또는 WAV 스크립트script.txt · `[제목]`, `[말씀]`, `[기도]` 필수 배경음악bgm.mp3 · 없으면 기본 배경음악 사용
파일명이 달라도 자동 분류를 시도하지만, voice.*와 bgm.* 규칙을 사용하면 음성 파일 충돌을 줄일 수 있습니다.
