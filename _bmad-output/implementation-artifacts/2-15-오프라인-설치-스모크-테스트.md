# Story 2.15: 오프라인 설치 스모크 테스트

Status: review

## Story

As a 개인 제작자,
I want 깨끗한 환경에서 오프라인 전체 흐름을 검증하고 싶다,
so that 설치본이 외부 서비스 없이 실제로 동작함을 확인할 수 있다.

## Acceptance Criteria

1. 네트워크가 차단된 깨끗한 Windows/macOS에서 설치 후 데이터 초기화, migration, binary 실행과 샘플 렌더링이 완료되며 Python·FFmpeg 별도 설치나 외부 전송이 필요 없다.

## Tasks / Subtasks

- [x] Windows x64와 macOS arm64 clean VM/runner의 네트워크 차단 절차를 고정한다. (AC: 1)
- [x] installer 설치→첫 실행→userData 초기화→migration→샘플 등록→render→폴더 열기 흐름을 자동화한다. (AC: 1)
- [x] system Python/FFmpeg가 없거나 PATH에서 제거된 상태를 확인한다. (AC: 1)
- [x] 네트워크 요청·DNS 의존·runtime download가 없음을 관측한다. (AC: 1)
- [x] artifact/log/checksum을 수집하고 플랫폼별 pass/fail 보고서를 작성한다. (AC: 1)

## Dev Notes

- 개발 머신의 캐시나 설치된 Python/FFmpeg 때문에 거짓 성공하지 않도록 clean image와 PATH를 증명한다.
- 테스트 fixture 외 사용자 콘텐츠를 수집하거나 외부로 전송하지 않는다.
- GUI 자동화가 불안정한 부분은 명확한 수동 검증 단계와 증거를 남기되 핵심 render 결과는 자동 probe한다.
