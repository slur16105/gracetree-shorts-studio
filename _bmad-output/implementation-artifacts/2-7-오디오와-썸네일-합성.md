# Story 2.7: 오디오와 썸네일 합성

Status: review

## Story

As a 개인 제작자,
I want 음성·BGM·썸네일을 영상에 합성하고 싶다,
so that 게시 형식에 필요한 모든 미디어를 한 결과에 포함할 수 있다.

## Acceptance Criteria

1. 배경 영상에 음성과 선택된 BGM을 포함하고 BGM 시작·종료 2초 fade를 적용하며 썸네일을 첫 프레임에 삽입한다.
2. FFmpeg는 검증된 실행 파일과 인자 배열만 사용하며 shell string과 원본 변경을 금지한다.

## Tasks / Subtasks

- [x] Story 1.7의 실제 BGM resolver 결과를 실행 스냅샷에서 사용한다. (AC: 1)
- [x] voice/BGM stream·duration과 thumbnail image를 ffprobe/decoder로 검증한다. (AC: 1)
- [x] BGM 2초 fade-in/out, 음성+BGM mix, 첫 프레임 thumbnail 합성 filter graph를 구현한다. (AC: 1, 2)
- [x] 공통 FFmpeg runner가 executable allowlist, args array, timeout, stderr redaction을 강제하게 한다. (AC: 2)

## Dev Notes

- 자동 음량 보정은 MVP 비목표다. 승인된 고정 gain/mix 규칙만 사용하고 임의 loudness normalization을 추가하지 않는다.
- 짧은 BGM에서 fade 구간이 겹치는 경우 결정적 clamp 또는 명시 오류를 정의한다.
- 썸네일 “첫 프레임”의 표시 duration과 이후 배경 전환을 fixture로 검증한다.
