# Story 2.11: 게시 가능 품질 게이트

Status: done

## Story

As a 제품 책임자,
I want 기준 영상과 생성 결과의 품질을 검증하고 싶다,
so that 자동화 결과가 실제 게시에 적합한지 판단할 수 있다.

## Acceptance Criteria

1. 최소 10개 기준 작업에서 자막 위치·한글·타이밍·전환·검은 프레임·음성·BGM을 자동·수동 평가하고 8개 이상이 추가 편집 없이 게시 가능해야 통과한다.
2. 실패 작업의 원인 범주와 보정 항목을 릴리스 품질 보고서에 남기고 결정적 자동 테스트와 장기 품질 게이트를 구분한다.
