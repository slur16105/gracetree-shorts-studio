# Story 2.6: 배경 영상 구성

Status: ready-for-dev

## Story

As a 개인 제작자,
I want 구간 길이에 맞는 배경 영상을 자동 구성하고 싶다,
so that 반복 영상 편집 없이 전체 영상 흐름을 만들 수 있다.

## Acceptance Criteria

1. 타이밍과 공통 영상으로 제목·말씀 영상을 구간 길이에 맞추고 기도 영상을 필요한 길이만큼 반복하며 반복 경계 약 0.5초 크로스페이드와 음성 종료 후 약 3초 여백을 적용한다.

## Tasks / Subtasks

- [ ] intro와 prayer-loop 리소스의 stream·duration·dimension을 ffprobe로 검증한다. (AC: 1)
- [ ] timing 구간을 기반으로 제목·말씀 속도 조정과 기도 반복 횟수를 결정적으로 계산한다. (AC: 1)
- [ ] 0.5초 crossfade와 3초 tail을 포함한 FFmpeg filter graph를 인자 배열로 생성한다. (AC: 1)
- [ ] 결과를 attempt temp에 만들고 duration·video stream·검은/정지 구간 기본 검사를 수행한다. (AC: 1)
- [ ] 짧은/긴/정확히 나누어지는 loop와 경계 프레임 통합 테스트를 추가한다. (AC: 1)

## Dev Notes

- 원본 공통 영상을 수정하지 않는다. 모든 중간 결과는 attempt temp에 둔다.
- shell string이나 사용자 값이 삽입된 filter 문자열을 만들지 않는다. filter graph 구성 요소를 검증된 값에서 생성한다.
- 크로스페이드로 전체 길이가 달라지는 계산을 테스트에서 명시한다.
